# Analytical Blog Writing / 深度分析写作

[English](#english) | [中文](#中文)

---

## English

Write in-depth analytical articles with data, research, and professional insights.

### What It Does

This skill helps you create data-driven, well-researched analytical content that demonstrates expertise and provides genuine value.

### Core Principles

1. **Data over opinion** - Support claims with evidence
2. **Depth over breadth** - Deep dive, not surface level
3. **Original research** - Bring new insights
4. **Clear methodology** - Explain how you reached conclusions
5. **Cite sources** - Give credit to original sources

### How to Use

Simply ask Claude to write an analytical article on your topic:

```
/analytical "How remote work affects productivity"
```

Or be more specific:

```
Write an analytical article about the impact of AI on software development,
including recent data from 2024-2025 and citing at least 5 sources.
```

### Article Structure

```markdown
# [Specific, Descriptive Headline]

## Executive Summary
3-4 sentence overview of findings

## The Question
What we investigated and why it matters

## Methodology
How we gathered and analyzed data

## Findings
Present data with charts/quotes

## Analysis
Interpret the findings

## Implications
What this means for the reader

## Conclusion
Key takeaways
```

### Example

```
You: /analytical The impact of four-day work weeks on employee productivity

Claude: [Generates a well-researched article with:
- Executive summary
- Data from recent studies
- Analysis of findings
- Citations to sources]
```

## 中文

撰写包含数据、研究和专业见解的深度分析文章。

### 功能说明

这个技能帮助你创建数据驱动、经过深入研究且展现专业见解的分析性内容。

### 核心原则

1. **数据优于观点** - 用证据支持主张
2. **深度优于广度** - 深入挖掘，而非浅层概述
3. **原创研究** - 带来新的见解
4. **清晰的方法论** - 解释如何得出结论
5. **引用来源** - 给予原始来源应有的认可

### 使用方法

直接让 Claude 撰写关于你主题的分析文章：

```
/analytical "远程办公如何影响生产力"
```

或者更具体一些：

```
撰写一篇关于 AI 对软件开发影响的分析文章，
包括 2024-2025 年的最新数据，并引用至少 5 个来源。
```

### 文章结构

```markdown
# [具体、描述性的标题]

## 执行摘要
3-4 句话的发现概述

## 研究问题
我们研究了什么以及为什么重要

## 研究方法
我们如何收集和分析数据

## 发现
用图表/引用呈现数据

## 分析
解读发现

## 启示
这对读者意味着什么

## 结论
关键要点
```

### 示例

```
你: /analytical 四天工作制对员工生产力的影响

Claude: [生成一篇精心研究的文章，包含：
- 执行摘要
- 最新研究数据
- 发现分析
- 来源引用]
```
