# Copywriter / 文案策划

[English](#english) | [中文](#中文)

---

## English

Write conversion-focused copy for products, marketing, UX, and sales pages.

### What It Does

Creates compelling copy that sells - focusing on benefits over features and using proven frameworks like AIDA and PAS.

### Core Principles

1. Focus on benefits, not features
2. Use active voice
3. Create urgency
4. Be specific with numbers
5. Write for scanning
6. Test different angles

### Copy Types

**Product Description**
Hook → Agitate → Solution → Benefits → Proof → CTA

**UX Writing**
- Buttons: "Get Your Free Guide" (not "Submit")
- Error messages: "Please enter a valid email"
- Form fields: "Email address" (not "Name")

**Sales Page**
Headline → Problem → Solution → Benefits → Social Proof → Offer → CTA

### How to Use

```
/copywriter product
[Describe your product and target audience]
```

```
/copywriter ux
[Describe the UI element and context]
```

```
/copywriter sales-page
[Describe your offer and target customer]
```

### Example

```
You: /copywriter product

I need a product description for a time-tracking app for freelancers.
Key features: automatic tracking, invoice generation, integrates with 50+ tools.

Claude: [Creates compelling copy focusing on benefits:
        "Get paid for every hour you work",
        "Stop losing money to forgotten billable time",
        with clear CTAs and social proof]
```

## 中文

为产品、营销、用户体验和销售页面撰写以转化为导向的文案。

### 功能说明

创建具有说服力的销售文案 - 关注利益而非功能，使用经过验证的框架如 AIDA 和 PAS。

### 核心原则

1. 关注利益而非功能
2. 使用主动语态
3. 创造紧迫感
4. 用具体数字说话
5. 便于扫描阅读
6. 测试不同角度

### 文案类型

**产品描述**
钩子 → 激化痛点 → 解决方案 → 利益 → 证明 → 行动号召

**UX 写作**
- 按钮："获取免费指南"（而非"提交"）
- 错误消息："请输入有效的电子邮件地址"
- 表单字段："电子邮件地址"（而非"姓名"）

**销售页面**
标题 → 问题 → 解决方案 → 利益 → 社会证明 → 优惠 → 行动号召

### 使用方法

```
/copywriter product
[描述你的产品和目标受众]
```

```
/copywriter ux
[描述 UI 元素和上下文]
```

```
/copywriter sales-page
[描述你的优惠和目标客户]
```

### 示例

```
你: /copywriter product

我需要为自由职业者时间追踪应用写产品描述。
关键功能：自动追踪、发票生成、与 50+ 工具集成。

Claude: [创建关注利益的吸引人文案：
        "获得你工作的每一小时报酬",
        "不再因忘记计费时间而损失金钱",
        包含清晰的 CTA 和社会证明]
```
