# Social Media Writer / 社交媒体写作

[English](#english) | [中文](#中文)

---

## English

Write engaging social media posts for Twitter/X, LinkedIn, Instagram, and more.

### What It Does

Creates scroll-stopping content with proper formatting, hashtags, and engagement hooks tailored for each platform.

### Supported Platforms

| Platform | Max Length | Style | Hashtags |
|----------|-----------|-------|----------|
| Twitter/X | 280 chars | Casual, concise | 2-3 |
| LinkedIn | 3-5 paragraphs | Professional, personal | 3-5 |
| Instagram | Short caption | Visual-first | 10-30 |
| Facebook | Medium length | Conversational | 5-10 |

### How to Use

**For Twitter:**
```
/tweet "I just shipped a side project"
```

**For LinkedIn:**
```
/linkedin "Share my article about remote work"
```

**For threads:**
```
/thread "Turn my blog post into a Twitter thread"
```

### Platform-Specific Tips

**Twitter/X:**
- Start strong - first line is crucial
- Use threads for longer content
- End with question or CTA
- Add image when relevant

**LinkedIn:**
- Write like you're talking to a colleague
- Share failures, not just successes
- Use bullet points for readability
- Post during business hours

### Example

```
You: /tweet "Just learned Rust after 10 years of JavaScript"

Claude: [Generates a relatable tweet with:
        - Strong hook
        - Personal insight
        - Relevant hashtag
        - Engagement question]
```

## 中文

为 Twitter/X、LinkedIn、Instagram 等平台撰写吸引人的社交媒体帖子。

### 功能说明

创建引人入胜的内容，具有正确的格式、话题标签和互动钩子，针对每个平台量身定制。

### 支持平台

| 平台 | 最大长度 | 风格 | 话题标签 |
|------|---------|------|----------|
| Twitter/X | 280 字符 | 随意、简洁 | 2-3 |
| LinkedIn | 3-5 段 | 专业、个人化 | 3-5 |
| Instagram | 短标题 | 视觉优先 | 10-30 |
| Facebook | 中等长度 | 对话式 | 5-10 |

### 使用方法

**Twitter:**
```
/tweet "我刚发布了一个副业项目"
```

**LinkedIn:**
```
/linkedin "分享我关于远程办公的文章"
```

**推文串:**
```
/thread "把我的博客文章转换成推文串"
```

### 平台专属技巧

**Twitter/X:**
- 开头要强 - 第一行至关重要
- 使用推文串表达较长内容
- 以问题或 CTA 结尾
- 相关时添加图片

**LinkedIn:**
- 像跟同事说话一样写作
- 分享失败，不仅仅是成功
- 使用要点提高可读性
- 工作时间发布

### 示例

```
你: /tweet "学了 10 年 JavaScript 后刚学会 Rust"

Claude: [生成一个相关的推文，包含：
        - 强有力的钩子
        - 个人见解
        - 相关话题标签
        - 互动问题]
```
