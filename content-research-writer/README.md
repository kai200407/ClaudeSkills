# Content Research Writer / 内容研究写作助手

[English](#english) | [中文](#中文)

---

## English

A collaborative writing partner that helps you research, outline, draft, and refine content while maintaining your unique voice.

### What It Does

1. **Collaborative Outlining** - Structure ideas into coherent outlines
2. **Research Assistance** - Find relevant information with citations
3. **Hook Improvement** - Strengthen openings to capture attention
4. **Section Feedback** - Review each section as you write
5. **Voice Preservation** - Maintain your writing style and tone

### How to Use

**Step 1: Create an outline**
```
Help me create an outline for an article about continuous discovery in product management.
```

**Step 2: Research and add citations**
```
Research recent data about AI's impact on product management and add citations.
```

**Step 3: Improve the hook**
```
Here's my introduction. Help me make the hook more compelling.
```

**Step 4: Get section feedback**
```
I just finished the "Why This Matters" section. Review it and give feedback.
```

**Step 5: Final review**
```
Review the full draft for flow, clarity, and consistency.
```

### Example Workflow

```
You: Help me create an outline for an article about remote work best practices.

Claude: [Creates structured outline with sections and research to-dos]

You: Research the section about team communication tools.

Claude: [Finds data, examples, and adds proper citations]

You: Here's my introduction... [paste draft]

Claude: [Provides feedback and suggests 3 improved hook options]
```

## 中文

协作式写作伙伴，帮助你研究、列大纲、起草和完善内容，同时保持你独特的写作风格。

### 功能说明

1. **协作大纲** - 将想法组织成连贯的大纲
2. **研究辅助** - 查找相关信息并添加引用
3. **开头优化** - 强化开头以吸引注意力
4. **逐段反馈** - 在写作时逐段审查
5. **风格保持** - 保持你的写作风格和语调

### 使用方法

**步骤 1：创建大纲**
```
帮助我为关于产品管理中持续发现的文章创建大纲。
```

**步骤 2：研究并添加引用**
```
研究关于 AI 对产品管理影响的最新数据并添加引用。
```

**步骤 3：改进开头**
```
这是我的介绍。帮我让开头更吸引人。
```

**步骤 4：获取段落反馈**
```
我刚写完"为什么这很重要"部分。审查并给出反馈。
```

**步骤 5：最终审查**
```
审查完整草稿的流畅度、清晰度和一致性。
```

### 示例工作流

```
你: 帮我为关于远程办公最佳实践的文章创建大纲。

Claude: [创建包含章节和研究待办事项的结构化大纲]

你: 研究团队沟通工具部分。

Claude: [查找数据、示例，并添加正确的引用]

你: 这是我的介绍...[粘贴草稿]

Claude: [提供反馈并建议 3 个改进的开头选项]
```
